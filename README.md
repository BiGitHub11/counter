# counter-app (Expo + TypeScript)

Minimal Expo React Native app scaffolded with the TypeScript template.

Prerequisites
- Node.js (16 or newer recommended)
- npm (comes with Node)
- Optional: Android Studio or a physical device with Expo Go

Install

```bash
npm install
```

Start development server

```bash
npx expo start
# or
npm run start
```

Run on Android (Windows)

```bash
npm run android
```

Run on web

```bash
npm run web
```

Notes
- To test on a physical device, install the Expo Go app and scan the QR code shown by the Metro dev server.
- If you don't have native toolchains, use the web or Expo Go for development.
